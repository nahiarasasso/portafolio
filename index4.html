Asecino de brujas

En este proyecto hice una página que trata de una trilogía de libros llamada "asecino de brujas". En esta página cuento de qué trata cada libro, un poco de la autora y explico cómo son los personajes principales.

Biblioteca virtual

Este proyecto es una biblioteca virtual donde puedes crearte un usuario, agregar títulos de libros o eliminarlos. También puedes buscar libros por título, autor y año de publicación.

Página de música

Este es un proyecto que trata de música. En esta página puedes crear un usuario para agregar o borrar canciones, buscar canciones por nombre o artista, y escucharlas.
